# WhatsApp Status Seen Bot

A Node.js-based WhatsApp bot to track status updates of contacts and stay online.

## Prerequisites
- Node.js v18+
- WhatsApp account
- Cloud hosting (e.g., Heroku, Render) for always-online functionality
